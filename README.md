### Como rodar o projeto

1. Clone o repositório e entre no diretório do projeto:
```bash
git clone git@github.com:Cursos-TI/desafio-xadrez-iugmali.git
cd desafio-xadrez-iugmali
```

2. Compile o projeto:
```bash
gcc xadrez.c -o xadrez
```

3. Execute o projeto:
```bash
./xadrez
```
